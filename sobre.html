Sobre o Site

Este site foi desenvolvido com o objetivo principal de treinar e aperfeiçoar conhecimentos em HTML, CSS e JavaScript, utilizando o framework Bootstrap para construção de interfaces responsivas e funcionais.
A temática escolhida foi o anime Diário de uma Apotecária, com foco especial na protagonista Maomao, uma personagem carismática e inteligente que atua como farmacêutica dentro da corte imperial. Como forma de tornar o aprendizado mais envolvente, o site apresenta uma seção de fanfiction, onde exibe algumas imagens relacionadas ao anime.
Além do conteúdo textual, o site inclui um carrossel de imagens construído com Bootstrap, exibindo várias ilustrações da personagem principal. Este recurso permite ao usuário visualizar diferentes representações da Maomao de forma interativa e visualmente agradável, sendo também uma forma prática de aplicar conceitos de front-end como manipulação de elementos, responsividade e estilização.
Este projeto serve como um exercício prático de desenvolvimento web.
